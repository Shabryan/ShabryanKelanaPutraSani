# Program Pengelola Data Mahasiswa

Program ini memungkinkan pengguna untuk mengelola data mahasiswa yang mencakup NIM, Nama, dan Nilai Akhir. Fitur utama dari program ini adalah:

1. Menambah data mahasiswa.
2. Menampilkan data mahasiswa.
3. Mencari data mahasiswa berdasarkan NIM.
4. Mengurutkan data mahasiswa berdasarkan Nilai Akhir secara menurun.
5. Menyimpan data mahasiswa ke dalam berkas.
6. Membaca data mahasiswa dari berkas.

### Instruksi Penggunaan:
1. Pilih "Tambah Data Mahasiswa" untuk memasukkan data baru.
2. Pilih "Tampilkan Semua Data" untuk melihat semua data.
3. Pilih "Cari Data Mahasiswa" untuk mencari berdasarkan NIM.
4. Pilih "Urutkan Data" untuk mengurutkan data berdasarkan Nilai Akhir.
5. Pilih "Simpan Data ke Berkas" untuk menyimpan data ke file.
6. Pilih "Baca Data dari Berkas" untuk memuat data dari file.

Data mahasiswa akan disimpan dalam file teks dengan format:
NIM, Nama, Nilai Akhir
